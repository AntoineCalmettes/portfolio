Green-drive

Description : Concevoir et réaliser la page d’accueil d’un site de e-commerce fictif. Le concept du site doit être imaginé (nom de l’entreprise, marchandises vendues, clientèle ciblée, etc.). Les marchandises vendues via le site doivent avoir une utilité sociétale (exemple : produits équitables, anti-gaspillage, programmes éducatifs, etc.).

Technologie : HTML5 et CSS3
